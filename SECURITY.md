# Security & Research Boundaries

This repository contains the **static public research and information site**
for the HADES initiative:

**Human-Assisted Diagnostic Evaluation System**

It does not contain:

- executable systems
- agent runtimes
- operational infrastructure
- production-connected code
- authentication surfaces
- data pipelines

There are no attack surfaces, credentials, or live integrations to protect.
